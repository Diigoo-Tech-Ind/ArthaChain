## ArthaChain Developer Portal v1

Information Architecture: Build → Run → Monetize → Govern

### Build
- EVM: Quickstart to deploy and call a contract via `/api/v1/contracts/*`.
- SDKs: `arthajs`, `arthapy` examples.

### Run
- Node setup, faucet, wallets, RPC endpoints.
- Storage (SVDB) upload/download/proofs.

### Monetize
- DealMarket: create deals, settle retrievals.
- Model/Dataset registry: publish and link code/data.

### Govern
- DAO: proposals, votes, execution.
- Token policy and roles.

See quickstarts in `DEV_PORTAL/quickstarts/` and API references in `DEV_PORTAL/api/`.
